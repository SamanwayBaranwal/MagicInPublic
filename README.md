# MagicInPublic

AI-powered startup idea analyzer that gives you raw, unfiltered reactions from real personas.

## What it does

Drop your startup idea and get reactions from:
- **Mom** — Your biggest hater & fan
- **The Builder** — Shipped 3 startups, got funded by YC, mass-quit twice
- **YC Partner** — Simulated Y Combinator partner analysis
- **Brutal Truth** — One line that hits hard
- **The Numbers** — Uniqueness, Demand, Difficulty, Stickiness scores

## Tech Stack

- Vanilla HTML/CSS/JS
- Node.js server
- AI via Hack Club API (Qwen model)
- html2canvas for card screenshots
- Space Mono + Space Grotesk fonts

## Run Locally

```bash
npm install
node server.js
```

Open `http://localhost:3000`
